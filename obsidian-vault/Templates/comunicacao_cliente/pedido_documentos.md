# Pedido de Documentos ao Cliente

## Regra geral

**Nunca pedir mais de 2-3 documentos por mensagem.** Dividir em várias mensagens se necessário, espaçadas no tempo. Indicar sempre o motivo de cada documento.

---

## Por tipo de caso

### Imigração — Primeira AR / Renovação

#### Mensagem 1 — Identificação
```
{nome}, para darmos entrada no vosso processo precisamos de começar pelos documentos de identificação:

1. 📄 Passaporte (foto da página com os dados e foto)
2. 📄 NIF (cartão ou documento das Finanças)

Podem enviar foto ou scan por aqui mesmo.
```

#### Mensagem 2 — Laboral
```
Obrigada! Agora precisamos dos documentos relacionados com o vosso trabalho:

1. 📄 Contrato de trabalho
2. 📄 Último recibo de vencimento

Se não tiverem o contrato em mãos, uma declaração da entidade patronal também serve.
```

#### Mensagem 3 — Segurança Social e morada
```
Estamos quase! Faltam apenas:

1. 📄 NISS (número da Segurança Social)
2. 📄 Comprovativo de morada (contrato de arrendamento ou atestado da junta)
```

#### Mensagem 4 — Registo criminal
```
Por último, vamos precisar do vosso registo criminal:

1. 📄 Registo criminal do país de origem (apostilado e traduzido)
2. 📄 Registo criminal português (se já vivem cá há mais de 1 ano)

O registo criminal português pode ser pedido online em: https://registocriminal.justica.gov.pt
```

### Imigração — Indicação SIS

```
{nome}, para podermos contestar a indicação SIS, precisamos de:

1. 📄 Passaporte (todas as páginas com carimbos)
2. 📄 Documento da AIMA com o indeferimento (se tiverem)

Vão existir outros documentos necessários depois, mas por agora estes são suficientes para começar.
```

### Nacionalidade

#### Mensagem 1
```
{nome}, para o processo de nacionalidade vamos precisar de:

1. 📄 Certidão de nascimento (apostilada e traduzida)
2. 📄 Passaporte (foto da página de dados)
```

#### Mensagem 2
```
Também precisamos de:

1. 📄 Registo criminal do país de origem (apostilado e traduzido)
2. 📄 Registo criminal português

Conseguem obter estes documentos?
```

#### Mensagem 3 (se por residência)
```
Como o pedido é por residência, precisamos ainda de:

1. 📄 Comprovativo de residência legal em Portugal (títulos de residência anteriores)
2. 📄 Comprovativo de conhecimento da língua portuguesa (se aplicável)
```

### Laboral

```
{nome}, para analisarmos o vosso caso laboral, precisamos de:

1. 📄 Contrato de trabalho (se existir)
2. 📄 Últimos 3 recibos de vencimento

Se tiverem qualquer correspondência com a entidade patronal (emails, cartas), guardem também — pode ser útil.
```

## Lembretes de documentos em falta

### Primeiro lembrete (3 dias)
```
{nome}, só para lembrar — ainda precisamos de:

📄 {documento_em_falta}

Conseguem enviar nos próximos dias? Se tiverem dificuldade em obter este documento, digam-nos que podemos ajudar.
```

### Segundo lembrete (7 dias)
```
{nome}, o vosso processo está a aguardar o seguinte documento:

📄 {documento_em_falta}

Sem este documento não conseguimos avançar. Precisam de ajuda para o obter?
```

### Terceiro lembrete (14 dias)
```
{nome}, já passaram duas semanas e o documento que precisamos ainda não chegou:

📄 {documento_em_falta}

Compreendemos que pode haver dificuldades. Se preferirem, podemos agendar uma chamada para resolver esta questão. O que acham?
```
